# Layanne Costa — Portfólio

Site de portfólio pessoal de Layanne Costa, Diretora de Arte baseada em Brasília.

🌐 [layannecosta.com.br](https://layannecosta.com.br)

## Sobre o projeto

Site single-page desenvolvido com HTML, CSS e JavaScript puro, com foco em performance, fidelidade visual e responsividade.

## Seções

- **Hero** — apresentação com foto e chamada principal
- **Marquee** — faixa animada com especialidades
- **Sobre** — trajetória e marcas atendidas
- **Serviços** — Direção de Arte, UI Design e Front-end
- **Trabalhos** — grid de projetos selecionados
- **Contato** — links de email, LinkedIn e portfólio

## Tecnologias

- HTML5
- CSS3 (variáveis, grid, animações)
- JavaScript vanilla
- Google Fonts — Space Grotesk + Inter
